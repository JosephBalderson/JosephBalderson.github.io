Mech 307: Mechatronics and Measurement Systems

Mechatronic and measurement system analysis and design; applied electronics; data acquisition; microcontroller interfacing and programming. This is an extension of ECE 204, where we learn the basics of electronics and learn more about PID tuning. This project was used to utilize these ideas and make an electrical or robotic project that had these controls. This project was the card dealer and used many things from what was learned in the class. This project was important for me to learn the value of making a mechatronics/electronics project. I ended up frying my laptop's mother board, and that was one of the most valuable lessons.

Mech 324: Dynamics of Machines

Dynamics of Machines is about the analysis and synthesis of moving machinery. The main focus of this class was 4-bar mechanisms. This uses the grashoff 4 bar equations to classify what kind of 4 bar is used. The project that I made was for a pill dispensor using a quick return mechanism. There are a lot of analytical calculations and iterative calculations that are done. To solve this, we use Matlab to solve the mechanisms, kinematics, and forces that are placed on the 4-bar.

Mech 344: Heat and Mass Transfer

Heat and Mass Transfer is about the thermal transport properties of substances, conduction, convection, radiation, transient heat transfer, numerical methods, and heat exchangers. This project uses nodal heat transfer to solve two problems with Python. This also includes an in-class example, which I solved and included the professor's code.
